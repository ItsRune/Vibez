addForPlayer​

This is a yielding function. When called, it will pause the Lua thread that called the function until a result is ready to be returned, without interrupting other scripts. Yields
since 2.3.7
</>
Widgets:addForPlayer(Player: Player) → ()

Adds a new player to show the widget to.

note

This method checks the player's policy to make sure games don't show widgets to users who aren't supposed to view them. Read up more on PolicyService to learn more!
