Settings

Note: You do not need all settings to make it work, every setting attached has a default value!

Our first version of settings were... Well, clunky and messy to say the least. So, we've decided to make it a lot more simple and easier to understand. We've also added a lot more settings to make it more customizable to your needs. Every setting below or on the right are each their own table. You can look at the examples below to see how to use them. If you have any questions, feel free to join our Discord and ask in any of the general channels. We'll be happy to help you out!

Example Usage​

local Vibez = require(game:GetService("ServerScriptService").Vibez)("API KEY", {
    Commands = {
        Enabled = true,
        MinRank = 200,
        -- Max rank is optional due to it being automatically set to, 255.
        Prefix = ";",
    },

    Interface = {
        Enabled = true,
        MinRank = 200,
    },
})

RankSticks

Enabled​

Toggles the rank sticks.
boolean false

Mode​

Determines the behavior the sticks should use. There's currently 2 modes; 'DetectionInFront' (Default) and 'ClickOnPlayer'. DetectionInFront checks for a player who's character is directly in front of you in order for it to work, unlike ClickOnPlayer where the player's character has to be clicked on by your mouse.

Cooldowns

Enabled​

Toggles whether different actions will have cooldowns associated with the specified mode.
boolean
false

Mode​

Determines how the cooldowns will behave:

  • PerTarget will add a cooldown to a specific target only.
  • PerStaff will add a cooldown to the staff member who used the action.
  • Both will add a cooldown to both the staff member and the target.

"PerTarget" | "PerStaff" | "Both"
PerTarget

Ranking​

number
